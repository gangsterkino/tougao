## 项目简介

这是一个在线投稿系统，支持不同角色的用户进行投稿、审核和管理等操作。系统包括 **非会员**、**会员**、**编辑** 和 **管理员** 四种角色，每个角色有不同的功能权限。前端使用 **React + TypeScript**，后端采用 **Python Flask**，数据库使用 **MySQL**。

## 技术栈

- **前端**：React + TypeScript + Umi
- **后端**：Python Flask
- **数据库**：MySQL
- **通信协议**：HTTP

## 功能概览

### 非会员
- 浏览刊物介绍、投稿须知等信息
- 注册成为会员

### 会员
- 修改个人密码
- 在线投稿（提交论文信息、Word 文档）
- 查询稿件状态（投稿、审稿、拒稿、通过）

### 编辑
- 修改个人密码
- 审核稿件并处理状态（“投稿” -> “审稿” -> “拒稿”/“通过”）

### 管理员
- 管理会员和编辑用户（增删改查）
